DESKTOP BILLING APPLICATION BUILD INSTRUCTIONS
==============================================

REQUIREMENTS:
- Python 3.7 or higher
- pip package manager
- Windows OS (for .exe creation)

INSTALLATION STEPS:
1. Install Python dependencies:
   pip install -r requirements.txt

2. Test the application:
   python main.py

3. Build executable with PyInstaller:
   
   OPTION 1 - Simple build:
   pyinstaller --onefile --windowed main.py

   OPTION 2 - Advanced build with custom settings:
   pyinstaller billing_app.spec

   OPTION 3 - Command line build with all options:
   pyinstaller --onefile --windowed --name "BillingSystem" --icon=icon.ico --add-data "*.db;." main.py

THERMAL PRINTER SETUP:
1. Install printer drivers
2. Connect printer via USB
3. Test printer connection in the application
4. For network printers, use IP address configuration

DEPLOYMENT:
1. Copy the generated .exe file from dist/ folder
2. The application will create billing.db automatically on first run
3. No additional installation required on target machines

TROUBLESHOOTING:
- If printer not detected, check USB connection and drivers
- For database issues, delete billing.db to reset
- Run as Administrator if file permission issues occur

FEATURES:
✅ Barcode scanner support (keyboard wedge)
✅ Thermal printer (ESC/POS 58mm)
✅ Offline SQLite database
✅ CSV export functionality
✅ Inventory management
✅ Bill history with search
✅ Customer information tracking
✅ Loose items with categories
✅ Live bill calculation
✅ Professional thermal receipts
✅ Admin settings with shop details management
✅ Optional admin authentication (default: admin/admin123)

PRINTER COMPATIBILITY:
- Epson TM-T20, TM-T82, TM-T88 series

- UPC-E

IMPORTANT NOTES (v2.1+):
------------------------
- All windows use `resize(width, height)` for safe default sizing. No more geometry warnings.
- All unsupported Qt stylesheet properties (like `transform`) have been removed. If you see 'Unknown property transform', check for dynamic stylesheet updates.
- When adding/editing inventory items, always enter the final price (including GST). Base price is auto-calculated.
- Bills and bill history show items in a compact table (Name, Qty, Base, SGST, CGST, Total).
- Bill summary shows average SGST% and CGST% instead of totals.

ADMIN SETTINGS (v2.2+):
-----------------------
- Default admin credentials: username=admin, password=admin123
- Shop details can be edited through Settings button in home dashboard
- Optional authentication can be enabled/disabled in admin settings
- When authentication is enabled, login screen appears before dashboard
- All changes to shop details require admin credentials verification

TROUBLESHOOTING (NEW):
----------------------
- **Unknown property transform**: All `transform:` properties have been removed. If you still see this warning, check for dynamic stylesheet updates in your code.
- **QWindowsWindow::setGeometry warnings**: All `setGeometry` calls have been replaced with `resize`. If you still see this warning, ensure you are not setting window sizes larger than your screen.
